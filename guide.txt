# AI Calendar 실행 가이드

이 프로젝트는 Frontend(ai-calendar)와 Backend(backend)로 구성되어 있습니다.
두 개의 터미널을 열어 각각 실행해야 합니다.

## 1. Backend 실행
1. `backend` 폴더로 이동합니다.
   cd backend
2. 의존성을 설치합니다 (최초 실행 시).
   npm install
3. Prisma 클라이언트를 생성합니다.
   npx prisma generate
4. 서버를 실행합니다.
   npm run dev

## 2. Frontend 실행
1. `ai-calendar` 폴더로 이동합니다.
   cd ai-calendar
2. 의존성을 설치합니다 (최초 실행 시).
   npm install
3. 개발 서버를 실행합니다.
   npm run dev

## 3. 접속
브라우저에서 http://localhost:5173 으로 접속합니다.





안드로이드 앱 빌드 가이드
안드로이드 프로젝트 생성이 완료되었습니다! 아래 순서대로 앱을 빌드해보세요.

1. 안드로이드 스튜디오에서 열기
터미널에 다음 명령어를 입력하면 안드로이드 스튜디오가 바로 열립니다:

npx cap open android
2. 에뮬레이터나 기기에서 실행하기
안드로이드 스튜디오가 프로젝트를 동기화(Gradle sync)할 때까지 기다립니다.
상단 메뉴에서 실행할 에뮬레이터를 선택하거나 안드로이드 폰을 연결합니다.
Run 버튼(초록색 재생 아이콘)을 눌러 앱을 실행합니다.
3. 출시용 파일 만들기 (APK/AAB)
플레이 스토어에 올리거나 직접 설치할 수 있는 파일을 만드는 방법입니다:

상단 메뉴에서 Build > Generate Signed Bundle / APK를 선택합니다.
Android App Bundle (플레이 스토어용) 또는 APK (직접 설치용)를 선택합니다.
Key Store를 새로 만들거나(비밀번호 잘 보관하세요!) 기존 키를 선택합니다.
Release 빌드 변형(variant)을 선택합니다.
Create를 누르면 파일이 생성됩니다.
TIP

코드 업데이트 방법 React 코드를 수정했다면, 아래 명령어를 실행해야 안드로이드 앱에도 반영됩니다:

npm run build
npx cap sync
